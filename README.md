# 投資ニュース自動リサーチシステム

海外の投資系YouTube動画を毎日自動収集・AI要約し、Word文書としてメール送信する自動化パイプラインです。

---

## システム概要

```
YouTube Data API
      ↓ (過去48時間の投資系動画を最大50件取得)
トランスクリプト取得 (youtube-transcript-api)
      ↓ (英語字幕 → なければ翻訳 → なければ概要欄にフォールバック)
Groq AI (LLaMA 3.3 70B) による日本語要約生成
      ↓ (1動画あたり約1000文字の詳細要約)
Word文書 (.docx) として output/ フォルダに保存
      ↓
Gmail でメール添付送信
```

---

## ニュース検索方法

### 検索対象
- **プラットフォーム**: YouTube
- **期間**: 実行時点から **過去48時間以内** に公開された動画
- **言語・地域**: 英語 (`relevanceLanguage=en`)、米国 (`regionCode=US`)
- **並び順**: 視聴回数順 (`order=viewCount`) → 最もトレンドの動画を優先

### 検索キーワード
```
"stock market investing Wall Street finance earnings"
```

### 動画コンテンツの取得優先順位
1. **英語の字幕（手動作成）**: `en`, `en-US`, `en-GB`, `en-AU`, `en-CA` の順に探索
2. **他言語字幕を英語に機械翻訳**: 英語字幕がない場合
3. **動画の概要欄 (description)**: 字幕が取得できない場合のフォールバック
4. **スキップ**: コンテンツが何も取得できない場合はその動画を除外

> テキストは最大10,000文字に切り詰めてAIに渡す。

### 有効動画の収集ロジック
- 最大50件の候補から上から順にチェック
- コンテンツが取得できた動画を **10件** 集まった時点で処理を開始
- 10件に満たない場合は集まった分で処理を続行

---

## AI要約の生成

### モデル
- **Groq API**: `llama-3.3-70b-versatile`

### 各動画の要約
- **文字数**: 約1000文字
- **言語**: 日本語
- **視点**: 証券アナリストの観点
- **含める内容**: 具体的な数値・銘柄名・市場背景・配信者の主張の根拠
- **除外する内容**: 動画内で語られていない独自の将来予測・波及効果の分析
- **Temperature**: 0.4（やや創造性を抑えて正確さ重視）

### ファイル名サマリーの生成
- レポート全体の内容を **15文字以内** で表現したファイル名を生成
- Windowsのファイル名として使用できる文字のみを使用
- 例: `イラン原油高とAI波及`、`エネルギーと技術の波及`

## 出力仕様

### ファイル形式
- **形式**: Word文書 (`.docx`)
- **保存先**: `output/` フォルダ（なければ自動作成）

### ファイル命名規則
```
{日付}_{連番}_{15文字サマリー}.docx
例: 20260403_01_米国株急落と利上げ.docx
```
- 同日に複数回実行した場合、連番が自動インクリメント (`_01_`, `_02_`, ...)

### ドキュメント構造
```
# 注目トピック　TOP10

## 注目トピック 第1位: {タイトル} (チャンネル名: {チャンネル})
（約1000文字の要約）

## 注目トピック 第2位: ...
...
```

### メール送信
- **件名**: `【自動リサーチ】海外投資YouTubeトップ10要約 ({ファイル名})`
- **宛先**: `GMAIL_ADDRESS`（自分自身に送信）
- **添付**: 生成した `.docx` ファイル
- **SMTPサーバー**: `smtp.gmail.com:465` (SSL)

## 処理フロー詳細

```
1. 環境変数の読み込み (.env または GitHub Secrets)
2. YouTube Data API で過去48時間の投資系動画を最大50件検索（視聴回数順）
3. 各動画のトランスクリプト（字幕）取得を試みる
   ├─ 英語字幕あり → そのまま使用
   ├─ 他言語字幕あり → 英語に翻訳して使用
   ├─ 字幕なし・概要欄あり → 概要欄テキストを使用
   └─ 全て取得不可 → その動画をスキップ
4. コンテンツ取得できた動画が10本集まった時点で次のステップへ
5. Groq AI (LLaMA 3.3 70B) で各動画を日本語で約1000文字に要約
   ※ APIレート制限対策で各動画の間に2秒のウェイト
6. 全要約をまとめて Groq AI でファイル名用15文字サマリーを生成
7. Word文書 (.docx) として output/ フォルダに保存
8. Gmail (SMTP SSL) でレポートをメール添付送信
```
